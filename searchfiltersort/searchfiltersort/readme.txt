== Installation ==

1. Upload the plugin folder to `/wp-content/plugins/searchfiltersort/`.
2. Activate the plugin through the **Plugins** menu in WordPress.
3. Open **Settings â†’ SearchFilterSort** to configure display options.
4. Insert the generated `[SearchFilterSort]` shortcode into any page or post.
   The shortcode value may change depending on the options you set in the plugin, so make sure to use the updated shortcode shown in the settings.


== Frequently Asked Questions ==

= Can I filter custom post types? =
Not at the moment. The plugin currently supports filtering default post types and WooCommerce products.
Support for additional custom post types may be introduced in future updates.

= Does it work with WooCommerce? =
Yes! The plugin can filter WooCommerce products by taxonomies, attributes, keywords, and more.

= Can I use multiple grids on a single page? =
Yes. Each shortcode instance can have its own parameters, so you can place multiple grids on one page without conflict.

= Can I change the text on the action button? =
Yes. The button label and alignment can be customized directly from the settings panel.

= How do I adjust the number of columns or filter position? =
Both options can be configured in the Layout Settings section of the plugin.
