== Frequently Asked Questions ==

= What does WPSyncSheets For Gravity Forms Lite do? =  
It automatically exports Gravity Forms entries to Google Sheets in real-time. Each submission is instantly added as a new row in your selected spreadsheet ‚Äî no manual export required.

= Do I need coding skills to use this plugin? =  
No. The plugin includes an easy-to-use interface with field mapping, auto spreadsheet creation, and real-time syncing ‚Äî no coding needed.

= Can I choose which Gravity Forms fields to export? =  
Yes. You can map only the fields you want to export using the plugin's column mapping feature.

= Will the plugin create a Google Sheet for me automatically? =  
Yes, during feed setup, the plugin can automatically create a new spreadsheet and sheet tab.

= Is the sync real-time or scheduled? =  
The free version supports real-time (run-time) syncing. Form entries are sent to Google Sheets immediately after submission.

= Can I export old Gravity Forms entries to Google Sheets? =  
The Lite version exports new entries. For bulk export of existing/old entries, upgrade to the [Pro version](https://www.wpsyncsheets.com/wpsyncsheets-for-gravity-forms/?utm_source=wp-repo&utm_campaign=wpsyncsheets-for-gravity-forms&utm_medium=faq).

= Does this plugin support importing entries from Google Sheets? =  
Import and bulk import functionality is available in the Pro version. It allows importing entries from Google Sheets back into Gravity Forms.

= Is this plugin compatible with other Gravity Forms addons? =  
Yes. The plugin is compatible with Gravity PDF, Gravity View, Gravity Forms Survey Add-On, WooCommerce Gravity Forms Product Add-On, Stripe, PayPal, Mollie, Razorpay, Dropbox, and more (via Pro version).

= Can I use this with multiple Gravity Forms? =  
Yes. You can create and configure separate feed settings for each form individually.

= How do I connect my Google Sheets account? =  
You need to generate a Google API Client ID, Client Secret, and Access Token via Google Cloud Console and add them to the plugin settings. Detailed instructions are available in the plugin documentation: https://docs.wpsyncsheets.com/wpssg-introduction/

== Installation ==

1. Upload the entire `wpsyncsheets-gravity-forms` folder to the `/wp-content/plugins/` directory.
1. Activate the plugin through the **Plugins** screen (**Plugins > Installed Plugins**).

You will find **WPSyncSheets Lite For Gravity Forms** menu in your WordPress admin screen.

For basic usage, have a look at the [plugin's Documentation](https://docs.wpsyncsheets.com/wpssg-introduction/).
